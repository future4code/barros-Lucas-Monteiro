Jogo BlackJack no JavaScript

Este projeto, tem como objetivo criar um joguinho básico em JavaScript, baseado no BlackJack. E como ele funciona?

Ao entrar, uma tela prompt, pergunta o nome do jogador, e começa a partida. Serão selecionada duas cartas para o jogador e duas para o computador, e serão mostrada no console. Cada carta tem um valor que serão somados. Quem tiver o maior número ganha. Será mostrado no console uma mensagem de quem ganhou e um prompt perguntando se quer outra jogada, se sim, recomeçará tudo novamente. Se não, mostrará uma mensagem no console informando que o jogo acabou.

Tecnologia utilizada:

  • JavaScript
